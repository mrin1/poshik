# 🐾 Poshik - Pet Care & Marketplace Platform

Welcome to **Poshik**, the ultimate all-in-one platform connecting Pet Owners with verified Veterinarians and Pet Shops. Built with modern web technologies, Poshik features real-time chat, instant appointment scheduling, and a fully-fledged e-commerce marketplace with live analytics.

## ✨ Key Features
* **Role-Based Access:** Unique dashboards tailored for Pet Owners, Doctors, Pet Shops, and Super Admins.
* **Real-time Chat & Consultations:** Instant messaging between owners and doctors using Supabase Realtime.
* **Smart Scheduling:** Doctors can manage active slots, accept/decline appointments, and set consultation fees.
* **Marketplace & Cart:** Owners can browse products, and Shops can track inventory, revenue, and daily analytics.
* **Optimized Performance:** Blazing fast data fetching and caching using React Query.

---

## 🛠️ Tech Stack
* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
* **State Management:** [Zustand](https://zustand-demo.pmnd.rs/) (Client Auth) & [React Query](https://tanstack.com/query/latest) (Server Data)
* **Backend & Database:** [Supabase](https://supabase.com/) (PostgreSQL, Auth, Realtime)
* **Icons:** [Lucide React](https://lucide.dev/)
